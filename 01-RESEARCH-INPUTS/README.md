# 01-RESEARCH-INPUTS

## Purpose
Raw research data collected during the competitive analysis phase.

## Contents
- `competitor-data/` - Raw competitor profile data
- `market-research/` - Market sizing, trends, and industry data
- `pricing-data/` - Competitor pricing information
- `feature-data/` - Feature comparison source data
- `sources.md` - Master list of all research sources

## When to Use
- During initial research phase
- When adding new competitor data
- When updating market information
- For fact-checking and source verification

## Related Directories
- `02-ANALYSIS-OUTPUTS/` - Processed analysis from this data
- `templates/competitive-analysis/01-RESEARCH-PHASE.md` - Research methodology guide
