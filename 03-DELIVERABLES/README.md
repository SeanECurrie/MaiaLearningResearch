# 03-DELIVERABLES

## Purpose
Client-ready deliverables only. This directory contains final, QA-passed documents.

## Contents
- `current/` - Latest version of all deliverables
  - `DOCX/` - Word documents
  - `HTML/` - HTML versions for web viewing
  - `PDF/` - PDF versions for distribution
  - `HTML/visualizations/` - Visualization HTML source files
  - `PDF/visualizations/` - Visualization PDFs (10 enterprise-themed)
- `v1.0/` - QA-corrected version (November 2025)
- `v0.9/` - Initial draft version (September-October 2025)
- `VERSION-HISTORY.md` - Complete version history and changelog

## When to Use
- When sharing deliverables with client
- When referencing current approved version
- When comparing versions
- For final client distribution

## File Naming Conventions
- DOCX: `executive-summary-CORRECTED.docx`, `full-report-CORRECTED.docx`
- PDF: `*-ENTERPRISE.pdf` (enterprise-themed visualizations)
- HTML: `*-viz.html` (visualization source files)
