## Methodology Note Format & Standards

All methodology notes follow consistent formatting:

### Visual Standards
- **Background:** #f5f5f5 (light gray for main box)
- **Border:** 2px solid #333 (dark gray)
- **Sub-sections:** White background (#ffffff), 3px left border #666
- **Typography:** H3 for main title, H4 for sub-calculations, 9pt for detailed bullets
- **Enterprise-appropriate:** Grayscale palette, no emojis, professional layout

### Content Standards
Each methodology note includes:
1. **Calculation basis:** Where numbers came from (research, client data, industry benchmarks)
2. **Formula/logic:** Explicit math shown (e.g., "15,000 schools × 10% = 1,500")
3. **Assumptions:** Key assumptions stated clearly
4. **Confidence level:** HIGH/MEDIUM/LOW with rationale
5. **Data sources:** Specific sources cited (NCES, ISC Research, client data, etc.)
6. **Limitations:** What's NOT included in estimates
